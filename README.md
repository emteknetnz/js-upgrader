# js-upgrader

Quick script to identify and update package.json files

## Installation

```bash
cd vendor
mkdir emteknetnz
cd emteknetnz
git clone git@github.com:emteknetnz/js-upgrader.git
cd js-upgrader
```

## Usage

Generate a report only output.txt

```bash
php run.php
```

Generate a report and update package.json files

```bash
php run.php update
```
